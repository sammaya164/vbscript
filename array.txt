<h2 id="redimステートメント">ReDimステートメント</h2>

<pre><code class="hljs language-vbs"><span class="hljs-keyword">Dim</span> a

</code></pre>
<p><code>ReDim</code>により配列の内容は初期化される。<br>
初期化したくない場合は<code>Preserve</code>をつける。</p>

<h3 id="split関数で作成した配列の要素は文字列として扱われる">Split関数で作成した配列の要素は文字列として扱われる。</h3>
<p>以下はArray関数とSplit関数で作成した配列について、各要素の総和を求める例。</p>
<pre><code class="hljs language-vbs"><span class="hljs-keyword">Dim</span> a, s

a = <span class="hljs-built_in">Array</span>(<span class="hljs-number">1</span>,<span class="hljs-number">2</span>,<span class="hljs-number">3</span>)    <span class="hljs-comment">'1,2,3の配列(数値)</span>
s = <span class="hljs-built_in">Split</span>(<span class="hljs-string">"1,2,3"</span>, <span class="hljs-string">","</span>) <span class="hljs-comment">'1,2,3の配列(文字列)</span>

MsgBox <span class="hljs-built_in">TypeName</span>(a(<span class="hljs-number">0</span>)) <span class="hljs-comment">'Integerと表示される</span>
MsgBox <span class="hljs-built_in">TypeName</span>(s(<span class="hljs-number">0</span>)) <span class="hljs-comment">'Stringと表示される</span>

MsgBox a(<span class="hljs-number">0</span>) + a(<span class="hljs-number">1</span>) + a(<span class="hljs-number">2</span>) <span class="hljs-comment">'6と表示される</span>
MsgBox s(<span class="hljs-number">0</span>) + s(<span class="hljs-number">1</span>) + s(<span class="hljs-number">2</span>) <span class="hljs-comment">'123と表示される</span>
</code></pre>
<p>配列sの総和(1+2+3=6)を求めたつもりが、文字列の結合(123)になってしまった。<br>
