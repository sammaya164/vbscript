<h2 id="filter関数">Filter関数</h2>
<p>Filter関数を使って、検索文字列を含む要素からなる配列を取得できる。</p>
<pre><code class="hljs language-vbs"><span class="hljs-keyword">Dim</span> fruit, tmp

fruit = <span class="hljs-built_in">Array</span>(<span class="hljs-string">"みかん"</span>, <span class="hljs-string">"りんご"</span>, <span class="hljs-string">"メロン"</span>)
tmp = <span class="hljs-built_in">Filter</span>(fruit, <span class="hljs-string">"ん"</span>) <span class="hljs-comment">'「ん」を含む要素を抽出</span>
MsgBox <span class="hljs-built_in">Join</span>(tmp, <span class="hljs-string">","</span>) <span class="hljs-comment">'みかん,りんごと表示される</span>
</code></pre>

<h2 id="for-each--nextステートメント">For Each ... Nextステートメント</h2>
<p>For Each ... Nextステートメントを使って配列の要素を順番に取得できる。</p>
<h3 id="1次元配列の場合">1次元配列の場合</h3>
<pre><code class="hljs language-vbs"><span class="hljs-keyword">Dim</span> fruit, buf

fruit = <span class="hljs-built_in">Array</span>(<span class="hljs-string">"みかん"</span>, <span class="hljs-string">"りんご"</span>, <span class="hljs-string">"メロン"</span>)

<span class="hljs-keyword">For</span> <span class="hljs-keyword">Each</span> buf <span class="hljs-keyword">In</span> fruit
  MsgBox buf <span class="hljs-comment">'みかん,りんご,メロンの順に表示される</span>
<span class="hljs-keyword">Next</span>
</code></pre>
<h3 id="2次元配列の場合">2次元配列の場合</h3>
<pre><code class="hljs language-vbs"><span class="hljs-keyword">Dim</span> fruit(<span class="hljs-number">1</span>,<span class="hljs-number">1</span>)
<span class="hljs-keyword">Dim</span> buf

fruit(<span class="hljs-number">0</span>,<span class="hljs-number">0</span>) = <span class="hljs-string">"みかん"</span>
fruit(<span class="hljs-number">0</span>,<span class="hljs-number">1</span>) = <span class="hljs-string">"りんご"</span>
fruit(<span class="hljs-number">1</span>,<span class="hljs-number">0</span>) = <span class="hljs-string">"メロン"</span>
fruit(<span class="hljs-number">1</span>,<span class="hljs-number">1</span>) = <span class="hljs-string">"ぶどう"</span>

<span class="hljs-keyword">For</span> <span class="hljs-keyword">Each</span> buf <span class="hljs-keyword">In</span> fruit
  MsgBox buf <span class="hljs-comment">'みかん,メロン,りんご,ぶどうの順に表示される</span>
<span class="hljs-keyword">Next</span>
</code></pre>
<p>上の例でbufに代入されるのは配列要素のコピーなので、この構文を使って元の配列を書き換えることはできない。</p>
