<p>前回確認した<code>Microsoft.ACE.OLEDB.12.0</code>を使ってデータベースを操作する。<br>
Access形式のデータベースファイルはMDBとACCDBがある。ここではACCDBを扱う。<br>
データベースを作成し、テーブルを追加する。</p>
<h2 id="accdbファイルを作成する">ACCDBファイルを作成する</h2>
<p>CatalogオブジェクトのCreateメソッドを使って、ACCESSのデータベースファイルを作成することができる。<br>
注) スクリプトファイルはShift_JISの文字コードで保存すること。</p>
<h3 id="方法1-connectionオブジェクトをcreateの引数にする">方法1: ConnectionオブジェクトをCreateの引数にする</h3>
<pre><code class="hljs language-vbs"><span class="hljs-comment">'データベースを作成する</span>

<span class="hljs-keyword">Dim</span> cat              <span class="hljs-comment">'Catalogオブジェクト</span>
<span class="hljs-keyword">Dim</span> con              <span class="hljs-comment">'Connectionオブジェクト</span>

<span class="hljs-keyword">Set</span> cat = <span class="hljs-built_in">CreateObject</span>(<span class="hljs-string">"ADOX.Catalog"</span>)
<span class="hljs-keyword">Set</span> con = <span class="hljs-built_in">CreateObject</span>(<span class="hljs-string">"ADODB.Connection"</span>)

con.ConnectionString = <span class="hljs-string">"Provider=Microsoft.ACE.OLEDB.12.0;Data Source=test.accdb;"</span>
cat.Create con
</code></pre>
<h3 id="方法2-接続文字列をcreateの引数にする">方法2: 接続文字列をCreateの引数にする</h3>
<pre><code class="hljs language-vbs"><span class="hljs-comment">'データベースを作成する</span>

<span class="hljs-keyword">Dim</span> cat              <span class="hljs-comment">'Catalogオブジェクト</span>

<span class="hljs-keyword">Set</span> cat = <span class="hljs-built_in">CreateObject</span>(<span class="hljs-string">"ADOX.Catalog"</span>)
cat.Create <span class="hljs-string">"Provider=Microsoft.ACE.OLEDB.12.0;Data Source=test.accdb;"</span>
</code></pre>
<p>後者の例ではConnectionオブジェクトが表に登場してないが、裏では作成されており、<br>
Createメソッドの戻り値として取得できる。</p>
<pre><code class="hljs language-vbs"><span class="hljs-comment">'データベースを作成する</span>

<span class="hljs-keyword">Dim</span> cat              <span class="hljs-comment">'Catalogオブジェクト</span>
<span class="hljs-keyword">Dim</span> con              <span class="hljs-comment">'Connectionオブジェクト</span>

con.Close <span class="hljs-comment">'接続を閉じる</span>
</code></pre>
<h2 id="テーブルを追加する">テーブルを追加する</h2>
<p>以上でデータベースファイル<code>test.accdb</code>が作成されたが、テーブルは存在してない。<br>
<code>elements</code>という名前でテーブルを追加してみる。</p>
<pre><code class="hljs language-vbs"><span class="hljs-comment">'テーブルを追加する</span>

<span class="hljs-keyword">Dim</span> cat <span class="hljs-comment">'Catalogオブジェクト</span>
<span class="hljs-keyword">Dim</span> tbl <span class="hljs-comment">'Tableオブジェクト</span>

<span class="hljs-keyword">Set</span> cat = <span class="hljs-built_in">CreateObject</span>(<span class="hljs-string">"ADOX.Catalog"</span>)
<span class="hljs-keyword">Set</span> tbl = <span class="hljs-built_in">CreateObject</span>(<span class="hljs-string">"ADOX.Table"</span>)

<span class="hljs-comment">'ACCDBファイルに接続する</span>
cat.ActiveConnection = <span class="hljs-string">"Provider=Microsoft.ACE.OLEDB.12.0;Data Source=test.accdb;"</span>

tbl.Name = <span class="hljs-string">"elements"</span> <span class="hljs-comment">'テーブル名を設定</span>
cat.Tables.Append tbl <span class="hljs-comment">'テーブルを追加</span>
</code></pre>
